= Modifier les informations d'identification de connexion vCenter pour le SCV
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Utilisez cette tâche pour mettre à jour les identifiants de connexion vCenter configurés dans le plug-in SnapCenter pour VMware vSphere. Ces identifiants sont utilisés pour accéder à vCenter. Si vous devez enregistrer ou réenregistrer SCV avec vCenter après un changement d'identifiant, effectuez cette tâche. Si le mot de passe vCenter change, désenregistrez ONTAP tools for VMware vSphere puis réenregistrez ONTAP tools avec le nouveau mot de passe afin que les sauvegardes vVol continuent de fonctionner.

.Avant de commencer
Vous devez connaître l'adresse IP et les informations de connexion pour l'interface utilisateur de gestion du SnapCenter Plug-in for VMware vSphere .  Vous devez également noter le jeton MFA généré à partir de la console de maintenance.

* L'adresse IP était affichée lors du déploiement du plug-in SnapCenter pour VMware vSphere.
* Utilisez les informations d'identification fournies lors du déploiement du plug-in SnapCenter pour VMware vSphere ou telles que modifiées ultérieurement.
* Générez un jeton MFA à 6 chiffres à l'aide des options de configuration du système de la console de maintenance.


.Étapes
. Connectez-vous à l'interface utilisateur de gestion du SnapCenter Plug-in for VMware vSphere .
+
Utiliser le format `\https://<appliance-IP-address>:8080`

. Dans le volet de navigation de gauche, sélectionnez *Configuration*.
+
image:scpivs44_image30.png["SnapCenter Plug-in for VMware vSphere"]

. Sur la page *Configuration*, dans la section *vCenter*, sélectionnez *Modifier*.
. Entrez le nouveau mot de passe, puis sélectionnez *Enregistrer*.
+
Ne modifiez pas le numéro de port.
